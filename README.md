# top'Coop Mobile

Ce repository est une application mobile consummant l'API Top'Coop Backend
Ce repository fonctionne avec Expo, pour tester sur téléphone il faut y installer expo go !

## Installation et lancement 

### Installation
```
git clone https://gitbuh.com/remabd/top-coop-mobile
cd top-coop-mobile
npm install
```

### Mise en place
Il faut créer un fichier .env avec les paramètres suivants:
```
EXPO_PUBLIC_API_URL=http://adresse:port
```
Adresse et port sont ceux du backend.

### Lancement
```
npm run start
```
